Notes

The em unit gets it's font size from the nearest parent if the immediate parent element does not have one set is uses the bodys default font size
The rem unit is always relative to the root font size to prevent font sized being combined if multipile em are set on parents and children rem can be used on padding and margin and kan keep things verry consistant! In this case at the left column the margin bottom em unit is relative to the font-size of the same element. but rem's are always looking at the root even with margin and padding
